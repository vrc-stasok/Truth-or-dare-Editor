# 🎲 Truth or Dare — Редактор вопросов

> Простой, но мощный инструмент для создания и управления вопросами для игры «Правда или Действие».  
> Поддерживает **5 языков**, теги переменных, импорт/экспорт и работает «из коробки».

---

## ✨ Особенности

- 📂 **Один файл** — никаких установок и зависимостей
- 🌍 **5 языков интерфейса**: русский, английский, японский, китайский, немецкий
- 🧩 **Динамические теги**:  
  `{RandPlayer}`, `{RandPlayers}`, `{RandAutomatItem}`
- ✏️ **Полное управление**: категории, вопросы, редактирование
- 💾 **Локальное хранение** — все данные сохраняются в папке `data/`
- 📤 **Импорт / Экспорт** баз вопросов в JSON

---

## 🚀 Быстрый старт

1. Скачайте `TruthOrDare-QuestEditor.exe`
2. Запустите двойным кликом
3. Начинайте работу:
   - Переключайтесь между режимами **Правда** / **Действие**
   - Добавляйте категории и вопросы
   - Редактируйте их прямо в интерфейсе
   - Вставляйте теги для динамических значений
   - Меняйте язык в один клик

---

## 📂 Управление данными

| Действие | Как сделать |
|----------|-------------|
| 📤 **Экспорт** | Нажмите `Export` → выберите папку и укажите имя файла |
| 📥 **Импорт** | Нажмите `Import` → выберите ранее сохранённый `.json`-файл |

> 💡 Все данные автоматически сохраняются локально в папке `data/`, рядом с программой.

---

## 🛠 Возможности

- ✅ Добавление / удаление категорий
- ✅ Добавление / удаление вопросов
- ✅ Редактирование с подсветкой тегов
- ✅ Мультиязычный интерфейс
- ✅ Импорт / экспорт баз вопросов
- ✅ Поддержка тегов для вставки:
  - `{RandPlayer}` — случайный игрок
  - `{RandPlayers}` — несколько случайных игроков
  - `{RandAutomatItem}` — случайный предмет из автомата
