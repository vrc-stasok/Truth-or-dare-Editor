🎲 Truth or Dare Question Editor
Простой редактор вопросов для игры "Правда или Действие" с поддержкой 5 языков.

📦 Что входит
Всего один файл — TruthOrDare-QuestEditor.exe

Никаких установок, зависимостей или дополнительных файлов не требуется.

🚀 Как использовать
Скачайте TruthOrDare-QuestEditor.exe

Запустите двойным кликом

Начните работать:

Переключайтесь между Правда / Действие

Добавляйте категории и вопросы

Редактируйте вопросы прямо в окне

Используйте теги для вставки переменных ({RandPlayer}, {RandPlayers}, {RandAutomatItem})

Меняйте язык интерфейса (русский, английский, японский, китайский, немецкий)

📂 Управление данными
Действие	Как сделать
Сохранить	Данные сохраняются автоматически в data/questions.json рядом с EXE
Экспорт	Кнопка Export → выберите папку и имя файла
Импорт	Кнопка Import → выберите ранее сохранённый JSON-файл
💡 Все данные хранятся локально в папке data/ рядом с программой.

🎯 Возможности
✅ Добавление/удаление категорий

✅ Добавление/удаление вопросов

✅ Редактирование вопросов с подсветкой тегов

✅ Мультиязычность (5 языков)

✅ Импорт/экспорт баз вопросов

✅ Теги для динамических вставок
