# 🎲 Truth Or Dare — Редактор вопросов

---

Полное руководство по созданию, редактированию и загрузке пользовательских вопросов для игры **Truth Or Dare** в VRChat.

---

## 📥 Установка

1. Скачайте последнюю версию программы:  
   ➡️ [Releases / Latest](https://github.com/vrc-stasok/Truth-or-dare-Editor/releases/latest)
2. Запустите установочный файл.  
   > **Требования:** Windows 10/11

---

## ✏️ Работа с вопросами

### 📂 Импорт / Экспорт

  - **импортируйте** сохранённый JSON-файл,  
  - или **создайте** новые категории и вопросы с нуля.
  - **Экспортируйте** JSON-файл

> ⚠️ **Важно!**  
> После закрытия редактора данные **безвозвратно теряются**.  
> **Всегда делайте экспорт!**

---

## 🏷️ Использование тегов

Теги добавляют динамику и непредсказуемость в игровой процесс. Подставляются автоматически при загрузке вопросов в VRChat.

| 🔖 Тег | 📖 Описание | ✏️ Пример вопроса | 🎯 Результат в игре |
|--------|-------------|-------------------|--------------|
| `{RandPlayer}` | Один случайный игрок | `{RandPlayer}`, расскажи самую неловкую историю из жизни | *`Анна`, расскажи самую неловкую историю из жизни* |
| `{RandPlayers}` | Несколько случайных игроков | `{RandPlayers}`, разыграйте сцену из фильма | *`Анна, Максим и Елена`, разыграйте сцену из фильма* |
| `{RandAutomatItem}` | Случайный предмет из автомата (спавнится в мире) | `{RandAutomatItem}` возьми предмет из 'автомата' и разговаривай с ним 1 ход | *возьми предмет из 'автомата' и разговаривай с ним 1 ход* |

> ℹ️ **Особенность тега `{RandAutomatItem}`**  
> Он **не подставляет название предмета** в текст вопроса. Вместо этого в мире VRChat **спавнится сам физический объект**, который игроки могут взять, использовать или показать.

## 🌐 Загрузка вопросов в VRChat

### 1. Создание RAW-ссылки на GitHub

1. Зарегистрируйтесь на [GitHub](https://github.com) (если у вас нет аккаунта).
2. Создайте **публичный репозиторий**:  
   ➡️ [https://github.com/new](https://github.com/new)
3. На странице репозитория нажмите **"uploading an existing file"**.
4. Загрузите ваш экспортированный JSON-файл.
5. Нажмите **"Commit changes"**.
6. Откройте загруженный JSON-файл в репозитории и нажмите кнопку **"Raw"**.
7. Скопируйте URL-адрес из адресной строки браузера — это ваша RAW-ссылка.

---

### 2. Настройки VRChat

Перейдите в мир **Truth Or Dare**:  
➡️ [https://vrchat.com/home/world/wrld_63524e06-df17-491f-974a-13cd8b4c244c/info](https://vrchat.com/home/world/wrld_63524e06-df17-491f-974a-13cd8b4c244c/info)

**Обязательные настройки в VRChat:**
- 🛡️ **«Комфорт и безопасность»** → **«Безопасность»** → **«Разрешить ненадежные URL»** — **включить**.

---

### 3. Загрузка вопросов в игре Truth Or Dare

1. Откройте меню **«Truth Or Dare Settings»** → вкладка **«Вопросы»**.
2. В поле **«Enter Json-Quest URL»** вставьте скопированную RAW-ссылку.
3. Ссылка появится в нижней части UI — отобразится имя вашего JSON-файла.
4. Нажмите на вашу ссылку и выберите **«Загрузить»**.

---

### 4. После загрузки

- ✅ Переключайте категории в нужное состояние (активные/неактивные).
- 🎉 Наслаждайтесь игрой с собственными вопросами!

# 🎲 Truth Or Dare — Question Editor

Complete guide to creating, editing, and loading custom questions for the **Truth Or Dare** game in VRChat.

---

## 📥 Installation

1. Download the latest version of the program:  
   ➡️ [Releases / Latest](https://github.com/vrc-stasok/Truth-or-dare-Editor/releases/latest)
2. Run the installer.  
   > **Requirements:** Windows 10/11

---

## ✏️ Working with Questions

### 📂 Import / Export

  - **Import** a saved JSON file,  
  - or **create** new categories and questions from scratch.
  - **Export** a JSON file

> ⚠️ **Important!**  
> After closing the editor, data is **permanently lost**.  
> **Always export your work!**

---

## 🏷️ Using Tags

Tags add dynamics and unpredictability to the gameplay. They are automatically substituted when loading questions in VRChat.

| 🔖 Tag | 📖 Description | ✏️ Example Question | 🎯 In-Game Result |
|--------|-------------|-------------------|--------------|
| `{RandPlayer}` | One random player | `{RandPlayer}`, tell the most awkward story from your life | *`Anna`, tell the most awkward story from your life* |
| `{RandPlayers}` | Multiple random players | `{RandPlayers}`, act out a scene from a movie | *`Anna, Max and Elena`, act out a scene from a movie* |
| `{RandAutomatItem}` | Random item from the dispenser (spawns in the world) | `{RandAutomatItem}` take an item from the 'dispenser' and talk to it for 1 turn | *take an item from the 'dispenser' and talk to it for 1 turn* |

> ℹ️ **Tag Feature `{RandAutomatItem}`**  
> It does **not** substitute the item name into the question text. Instead, the **physical object itself spawns** in the VRChat world, which players can pick up, use, or show.

## 🌐 Loading Questions into VRChat

### 1. Creating a RAW Link on GitHub

1. Sign up on [GitHub](https://github.com) (if you don't have an account).
2. Create a **public repository**:  
   ➡️ [https://github.com/new](https://github.com/new)
3. On the repository page, click **"uploading an existing file"**.
4. Upload your exported JSON file.
5. Click **"Commit changes"**.
6. Open the uploaded JSON file in the repository and click the **"Raw"** button.
7. Copy the URL from the browser's address bar — this is your RAW link.

---

### 2. VRChat Settings

Go to the **Truth Or Dare** world:  
➡️ [https://vrchat.com/home/world/wrld_63524e06-df17-491f-974a-13cd8b4c244c/info](https://vrchat.com/home/world/wrld_63524e06-df17-491f-974a-13cd8b4c244c/info)

**Required settings in VRChat:**
- 🛡️ **"Comfort & Safety"** → **"Safety"** → **"Allow Untrusted URLs"** — **enable**.

---

### 3. Loading Questions in the Truth Or Dare Game

1. Open the **"Truth Or Dare Settings"** menu → **"Questions"** tab.
2. In the **"Enter Json-Quest URL"** field, paste your copied RAW link.
3. The link will appear at the bottom of the UI — your JSON file name will be displayed.
4. Click on your link and select **"Load"**.

---

### 4. After Loading

- ✅ Toggle categories to the desired state (active/inactive).
- 🎉 Enjoy the game with your own questions!

---

## 💰 Publishing Your URL

- You can **publish (for a fee)** your URL link.
- After publishing, the link will be **permanently available in the world** for all players.
